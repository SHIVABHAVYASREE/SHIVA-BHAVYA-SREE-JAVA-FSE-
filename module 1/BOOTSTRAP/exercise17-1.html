Bootstrap Icons is a free, high quality, open source icon library with over 1,600 icons. It's designed to work with Bootstrap components but can be used with any project.

How to Use Bootstrap Icons

  1. Include the Bootstrap Icons CSS in your project
  2. Use the <i> element with the appropriate classes
  3. Customize size, color, and other properties with CSS

Note

Bootstrap Icons can be used in various ways: as web fonts, as SVGs, or as individual SVG files.
